## Proyek Pilihan

### Inventory Manager
- **Deskripsi**: Sistem manajemen persediaan untuk UKM  
- **Tech Stack**: React + TypeScript, Node.js, PostgreSQL, Docker  
- **Peran**: Desain arsitektur, fitur order & sync stok, optimasi kueri

### Realtime Chat Platform
- **Deskripsi**: Aplikasi chat dengan presence & notifikasi real-time  
- **Tech Stack**: Next.js, WebSocket, Redis, MongoDB  
- **Peran**: Implementasi realtime messaging, skalabilitas koneksi, monitoring  
- **Hasil**: Latensi rata-rata <100ms pada beban normal.

- **Tech Stack**: NestJS, PostgreSQL, S3, GitHub Actions  
- **Peran**: Bounded contexts, integrasi pembayaran, deployment otomatis  
- **Hasil**: Mempercepat time-to-market fitur checkout baru dari 3 minggu → 4 hari.

## Quick Start (Umum)
```bash
# Clone repository
git clone https://github.com/adull01/repo.git
cd repo

# Install dependencies
pnpm install   # atau npm install / yarn

# Konfigurasi environment
cp .env.example .env
# Edit .env dengan konfigurasi lokal

# Jalankan development
pnpm dev

# Jalankan test
pnpm test

# Build production
pnpm build
```
